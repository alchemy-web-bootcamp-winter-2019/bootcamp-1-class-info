Team File
===

Files are about **content**

# First Recipe

```sh
> pwd
> ls
> touch empty-file.txt
> ls
> cat empty-file.txt
```

```sh
> ls
> rm empty-file.txt
> ls
```

# Second Recipe

```sh
> pwd
> ls
> echo "text in a file" > content-file.txt
> ls
> cat content-file.txt
```

```sh
> ls
> echo "change text in a file" > content-file.txt
> ls
> cat content-file.txt
```

```sh
> ls
> rm content-file.txt
> ls
```
